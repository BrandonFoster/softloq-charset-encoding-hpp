Detailed Description

Follows the behavior stated here: (https://infra.spec.whatwg.org/#code-points). A code point is a Unicode code point and is represented as "U+" followed by four-to-six ASCII upper hex digits, in the range U+0000 to U+10FFFF, inclusive. A code point’s value is its underlying number.
